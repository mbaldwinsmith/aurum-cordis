Usage

All symbols are stored as standalone SVG files in assets/icons/. They use stroke="currentColor" and fill="none" (except filled details), so they inherit any CSS color property. Embed with <img src="../assets/icons/sulphur.svg" alt="…"/> for automatic scaling, or inline for full CSS control.
